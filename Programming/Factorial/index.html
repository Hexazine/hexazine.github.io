高速階乗計算

\(\prod_{i=1}^k(n+i)\ \mathrm{mod}\ p\) を \(O(\sqrt k\log k)\) で計算するよ すごいね
\(n=0\) のとき \(k!\ \mathrm{mod}\ p\) になるよ

方針

仕方ないので、一から復習することにした

\(f_d(x):=\prod_{i=1}^d(x+i),\ s=[\sqrt k]\) とすると、 \[\prod_{i=1}^k(n+i)=\left(\prod_{i=0}^{s-1}f_s(n+si)\right)\prod_{i=s^2+1}^{k}(n+i)\] となるが、右側は愚直に \(O(\sqrt k)\) で計算できるからよくて、左側を「\(s\) 次の多項式の \(s\) 点での値の積」と捉えられると multipoint evaluation を思いつくので、やってみる

multipoint evaluation 解

Q. multipoint evaluation とは？
A. ある \(k\) 次多項式があって、これの \(n\) 点での値全てを求めるやつ
　この記事では重要ではないので詳細な説明は省くが、因数定理をベースとした考え方で、「多項式を多項式で割った余りを \(O(n\log n)\) で求めるアルゴリズム」を使って \(O(n\log^2 n+k\log k)\) で求めることができる。
　\(f_s(x)\) は分割統治法を用いて乗算 \(O(s\log s)\)、計算回数 \(O(\log s)\) で計 \(O(s\log^2s)\) で求めることができ、\(O(s\log^2s)\) で multipoint evaluation を実行することで \(O(s\log^2s)=O(\sqrt k\log^2k)\) で求まる。
　しかし評価点が公差 \(s\) の等差数列を成していることに気づけば更に高速に求めることができる。

Lagrange Interpolation

ある \(d\) 次多項式 \(g(x)\) の \(d+1\) 点での値 \(g(x_0),\ g(x_1),\ \cdots\ ,\ g(x_d)\) が与えられているとき、 \[g(x)=\sum_{i=0}^dg(x_i)\prod_{j=0,\ j\neq i}^d\frac{(x-x_j)}{(x_i-x_j)}\] が成り立つ\(^{\ast 1}\)。この式を使って元の多項式 \(g(x)\) やある点での値 \(g(a)\) を求めることを Lagrange Interpolation(ラグランジュ補間)という。
　ところで、\(g(0),\ g(1),\ \cdots\ ,\ g(d)\) が与えられているとき、 \[\begin{aligned}g(m+k)&=\sum_{i=0}^dg(i)\prod_{j=0,\ j\neq i}^d\frac{m+k-j}{i-j}\\&=\sum_{i=0}^dg(i)\left(\frac{\prod_{j=0}^d(m+k-j)}{m+k-i}\cdot\frac{1}{\prod_{j=0}^{i-1}(i-j)\cdot\prod_{j=i+1}^d(i-j)}\right)\\&=\left(\prod_{j=0}^d(m+k-j)\right)\left(\sum_{i=0}^dg(i)\cdot\frac{1}{m+k-i}\cdot\frac{1}{i!\cdot(-1)^{d-i}(d-i)!}\right)\\&=\left(\prod_{i=0}^d(m+k-i)\right)\left(\sum_{i=0}^d\frac{g(i)}{(-1)^{d-i}i!(d-i)!}\cdot\frac{1}{m+k-i}\right)\end{aligned}\] がいえる。これを用いて \(g(m),\ g(m+1),\ \cdots\ ,\ g(m+e)\) を求めるとき、左側は固定された値を \(O(d)\) で1回だけ求めればよいが、右側の値は \(k\) があって変わり続けるので \(O(d)*O(e)=O(de)\) かかりそうである。しかし、 \[a(x):=\frac{g(x)}{(-1)^{d-x}x!(d-x)!},\ b(x):=\frac{1}{m-d+x}\] とおくと、 \[\sum_{i=0}^d\frac{g(i)}{(-1)^{d-i}i!(d-i)!}\cdot\frac{1}{m+k-i}=\sum_{i=0}^da(i)b(k+d-i)\] となり、畳み込みになっていることが分かる。よって、\(a(0),\ a(1),\ \cdots\ ,\ a(d)\) 及び \(b(0),\ b(1),\ \cdots\ ,\ b(2d)\) を \(O(d)\) で求めておけば、求めたいものである \[a(0)b(d)+a(1)b(d-1)+\ \cdots\ +a(d)b(0),\]\[\ a(0)b(d+1)+a(1)b(d)+\ \cdots\ +a(d)b(1),\]\[\vdots\]\[a(0)b(2d)+a(1)b(2d-1)+\ \cdots\ +a(d)b(d)\] は任意mod畳み込みで \(O(d\log d)\) で求められる。よって \(g(m),\ g(m+1),\ \cdots\ ,\ g(m+d)\) を \(O(d\log d)\) で求められた。
　\(g(n),\ g(n+s),\ \cdots\ ,\ g(n+sd)\) が与えられているとき、\(d\) 次多項式 \(h(x):=g(sx+n)\) の \(d+1\) 点での値 \(h(0),\ h(1),\ \cdots\ ,\ h(d)\) が与えられているので、\(h(\frac{m-n}{s}),\ h(\frac{m-n}{s}+1),\ \cdots\ ,\ h(\frac{m-n}{s}+d)\) を補間することで \(g(m),\ g(m+1),\ \cdots\ ,\ g(m+d)\)
